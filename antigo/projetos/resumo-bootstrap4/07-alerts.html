Bootstrap 4 - Alerts

Alerts no Bootstrap 4

O Bootstrap 4 oferece classes que cria um alerta facilmente. Estas são: .alert .alert-* .alert-warning .alert-danger .warning-primary

Sucesso! Este é um alerta de sucesso!
Warning! Este é um alerta de aviso

.alert-link

A class .alert-link é usado para estilizar um elemento <a> de acordo com o tipo de alerta.

Perigo! Exemplo de alerta com link

Alerta fechável

Para criar um alerta fechável, você precisa adicionar a class .alert-dismissible no container de alert e a class .close num link ou botão + o atributo data-dismiss="alert"

Observação!

Para esta classe funciona, os scripts JQuery, popper e bootstrap precisam estar importados (em ordem respectiva).
