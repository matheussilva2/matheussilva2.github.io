Bootstrap4 - Badges

Badges Básicos

Bagdes são usados para adicionar alguma informação adicional no conteúdo. Para fazer isto, use a class .badge e .badge-*

Badges Contextuais

Os badges contextuais são definidos no .badge-*. São usadas os mesmos sufixos de classes que vimos anteriormente (primary, secondary, success, danger, warning, info, light, dark)

Pill Badges

Estes são a mesma coisa que os anteriores, mas mais arredondados. Para usar, aplique a class .badge-pill
