O Conjunto Collapse

Esta class é muito útil para esconder um grande parágrafo que será mostrado quando clicado em um botão ou link.

A class .collapse indica o elemento será colapsado, neste caso a nossa div. Para esconder ou mostrar o conteúdo, criamos um button ou a com o atributo data-target="#id" colocando o id da div que sera colapsada e também data-toggle="collapse".

Também podemos usar href nos elementos <a> para substituir o data-target

Neste caso usamos o atributo data-parent="#id" no card.
